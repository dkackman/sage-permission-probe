# Exploit Tester Sage App

This is a Sage app to test exploits. It is not meant to be used for anything else. It is not meant to be used in production. It is not meant to be used by anyone other than me. It is not meant to be used at all.

1. Sage sdk is expected to be at `../../Hadamcik/sage/packages/sage-app-sdk`
2. `pnpm install`
3. `pnpm build`
4. `pnpm preview`
5. Install to Sage by URL
