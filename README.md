# Example Sage App

1. Have [my Sage fork](https://github.com/Hadamcik/sage/tree/apps) cloned and checked out as a sibling directory
2. `pnpm install`
3. `pnpm build`
4. `pnpm preview`
5. Install to Sage by URL
